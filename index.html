Frank

Frank is a simple, domain specific language in F# for writing web applications. Frank is inspired by the Ruby dynamic duo of Rack and Sinatra and aims to provide the simplicity of those frameworks to .NET web developers. Frank also gives a nod to the Snap framework since it beat Frank to the punch on using the state monad.

As of right now, Frank will run on top of both Frack and the WCF Web APIs using the Microsoft.Http libraries.

Usage

Define a Frank application:

> let myApp = FrankApp.init [
    >   get "/" (putText "Hello world!")
    >   post "/order" (fun () -> frank {
    >     let! p = getParams
    >     createThingFromParams p
    >     do! redirectTo "/" })
    >   // More to come...
    > ]
    val myApp : Func<HttpRequestMessage,HttpResponseMessage>
